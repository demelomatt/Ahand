# Ahand
![logoahand](logofull.png)
Descrição:
    Projeto para automação de rotinas que envolvem arquivos PDF.
    Transformar arquivos PDF em legíveis, pesquisar por palavras dentro do PDF, e se encontrar, executar alguma ação.
